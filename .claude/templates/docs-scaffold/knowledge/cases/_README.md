# docs/knowledge/cases/ — 重要案件詳細記録

このフォルダには、`/sf-memory cat6`（`sf-analyst-cat6` エージェント）が自動生成する**重要案件の詳細記録ファイル**が格納される。

## 生成条件

以下のいずれかを満たす Backlog 完了課題が対象（直近 50 件上限）:

- `actualHours >= 2h`（実績工数 2 時間以上）
- `docs/knowledge/pitfalls.md` 検出条件に合致（ハマりポイントあり）
- 課題本文 + コメント合計 1,500 文字以上

## ファイル構成（各 `{issueKey}.md`）

```
## TL;DR（200字以内）
## 症状・要件
## 調査・検討の経緯
## 採用方針
## 却下案・代替案
## 教訓・再発防止
## 関連リンク
```

## 運用ルール

- `/backlog` 完了時（Phase 3.8）にも 1 件ずつ自動生成される
- 同名ファイルが既に存在する場合はスキップ（上書きしない）
- 手動でファイルを追加・編集することも可能（自動生成時に上書きされない）
- `docs/knowledge/case-index.md` の「詳細」列からこのフォルダへリンクされる

## 索引との関係

| 索引ファイル | 役割 |
|---|---|
| `case-index.md` | 全課題の 1 行サマリ（症状/根本原因/採用方針/教訓） |
| `pitfalls.md` | 再発防止すべきハマりポイント抽出 |
| `cases/{issueKey}.md` | 重要案件の詳細記録（本フォルダ） |
