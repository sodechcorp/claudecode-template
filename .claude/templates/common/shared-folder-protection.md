# 共有フォルダ・社内フォルダの保護

## ルール概要

| 対象 | 削除 | 書き込み |
|---|---|---|
| `G:\共有ドライブ` / `G:\Shared drives` | hook ハードブロック（bypass 不可） | 技術的ブロックなし → 下記確認フローを必ず通す |
| ネットワークドライブ（`\\server\...`）| 行動指示のみ | 行動指示のみ |
| SharePoint / OneDrive | 行動指示のみ | 行動指示のみ |

## 書き込み前の確認フロー

書き込み前に、**AskUserQuestion ではなく日本語の地の文**で以下を出してユーザーが「OK」「進めて」等で明示的に承認するまで待つ:

```
⚠️ `G:\共有ドライブ` 配下のファイル `{パス}` に書き込みます。共有データのためチーム全員に影響します。実行してよろしいですか？
```

## 禁止事項

- `C:/tmp/` 等を経由する回避策での書き込みは禁止。直接書き込む or 諦める
- 選択 UI（AskUserQuestion）で承認を取るのは禁止。自然文の会話で確認する
